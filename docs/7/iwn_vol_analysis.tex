The portfolio holds \textbf{Grayscale BTC Mini Trust}
(ticker BTC, \$32.35/share), representing 46.1\% of portfolio value.

  \item \textbf{Direct put protection remains prohibitive}: at 81.5\% realized
        vol, Grayscale BTC puts cost 14\% of spot per quarter.
\end{itemize}

\subsection{Correlation Divergence: Grayscale vs.\ Spot}

The dashboard (runtime service) computes rolling return correlations across
7 windows (5--35 trading days). The method is:
\[
  \rho_{N}(t) = \frac{\sum_{i=1}^{N}(r^{A}_i - \bar{r}^{A})(r^{B}_i - \bar{r}^{B})}
  {\sqrt{\sum(r^{A}_i - \bar{r}^{A})^2}\;\sqrt{\sum(r^{B}_i - \bar{r}^{B})^2}}
  \quad\text{where } r_i = \ln\!\left(\frac{P_i}{P_{i-1}}\right)
\]

This is the \textbf{correct} Pearson correlation on log returns.

\subsection{Implication for Hedging}

Since Grayscale BTC moves with equities:

\begin{itemize}[nosep]

        closest proxy) is $+0.77$ correlated with the trust.
  \item \textbf{GLD is the only potential hedge}: negative to low correlation
        with both Grayscale BTC and equities.
\end{itemize}
