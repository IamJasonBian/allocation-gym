\section{Executive Summary}
% ============================================================

We evaluate IWN (iShares Russell 2000 Value) as a BTC-decorrelated hedge
allocation, benchmarked against SPY for cost efficiency. Key findings:

\begin{itemize}[nosep]
  \item \textbf{SPY is the cheaper hedge}: 3M ATM puts cost \textbf{2.59\%}
        of spot vs.\ IWN's 4.00\%, and SPY carries \textit{stronger} negative
        BTC correlation at the 1M/3M horizon ($-0.112$ vs $-0.028$).
  \item \textbf{IWN adds diversification}: Near-zero BTC correlation across
        all windows makes IWN a \textit{pure} diversifier, not a directional
        BTC hedge. Useful when BTC sentiment is unclear.
  \item \textbf{Recommended reallocation}: 8--12\% of portfolio (\$15--23K)
        split between SPY puts (directional BTC hedge) and IWN equity
        (decorrelated return). Total quarterly hedge cost: \$400--800.
  \item \textbf{Bullish BTC variant}: When long BTC with conviction, the same
        IWN allocation acts as a non-correlated return source that doesn't
        drag on BTC upside---unlike SPY/QQQ which move against BTC tactically.
\end{itemize}

% ============================================================

Prior analysis used 3M/6M/12M trailing windows. We now focus on
\textbf{1-month (21-day) rolling correlation}, which captures the tactical

\subsection{Key Observations}

\begin{enumerate}[nosep]
  \item \textbf{SPY and QQQ are more negatively correlated with BTC} than
        IWN at the 3M horizon ($-0.112$ and $-0.135$ vs $-0.028$). This
        negative correlation strengthens at shorter windows---at the 1M
        rolling level, large-cap indices tend to diverge further from BTC
        during risk-off rotations.
  \item \textbf{IWN is correlation-neutral across all windows}: it never
        exceeds $|\rho| = 0.06$. This makes it a \textit{diversifier} rather
        than a \textit{hedge}---it won't move against BTC, but it also won't
        move with it.
  \item \textbf{IWM and IWO (growth) are positively correlated}: Russell 2000
        blend/growth track BTC modestly, especially at 6--12M ($+0.06$ to
        $+0.09$). Avoid these as BTC hedges.
  \item \textbf{Correlation is regime-dependent}: All tickers show a
        negative-to-positive drift from 3M to 12M, suggesting recent
        tactical divergence from a longer-term macro co-movement.
\end{enumerate}

\subsection{Rolling 1M Correlation Framework}

The existing pipeline computes $\rho_{63d}$, $\rho_{126d}$, $\rho_{252d}$.
For the 1M tactical signal, we add a 21-day window:
\[
  \rho_{1M}(t) = \mathrm{corr}\bigl(\{r^{\mathrm{BTC}}_i\}_{i=t-20}^{t},\;
                  \{r^{\mathrm{asset}}_i\}_{i=t-20}^{t}\bigr)
\]
where $r_i = \ln(P_i / P_{i-1})$. This will be available from the Redis
feed once deployed, using the \texttt{bars} field in the \texttt{OptionsRecord}
proto. For now, the 3M snapshot serves as a conservative proxy.

\subsection{Sizing Rationale}

  \item \textbf{SPY puts (\$12K notional, \$311/quarter)}: We already hold
        \$9.3K SPY + \$8.5K VOO = \$17.7K S\&P~500. Buying puts on
        existing SPY shares costs only \textbf{0.16\% of portfolio per quarter}
        (0.64\% annualized). SPY's $-0.112$ BTC correlation means the puts
        gain value precisely when BTC sells off.
  \item \textbf{IWN equity (\$7.5K, {\raise.17ex\hbox{$\scriptstyle\sim$}}38 shares)}: Adds a
        non-correlated return source. No put overlay needed on IWN itself---its
        low vol (17.8\%) and strong 1Y return (+34.75\%) make the equity
        position self-hedging via value-factor mean reversion.
  \item \textbf{Total cost}: \$311/quarter = \textbf{0.16\%} of portfolio.
        At 4 quarters/year, annual hedge cost is \textbf{0.64\%}---well within
        the 1--2\% budget typical for tail-risk overlays.

\textbf{When bullish on BTC}, the IWN component is especially attractive:

\begin{itemize}[nosep]
  \item IWN's near-zero correlation ($\bar{\rho} = -0.002$) means it
        \textbf{doesn't drag on BTC rallies}. Unlike SPY ($\rho = -0.11$),
        which tends to weaken when BTC surges, IWN moves independently.
  \item The \textbf{value premium} in small-cap value (+34.75\% 1Y) provides
        a return floor even in BTC-flat environments.
  \item \textbf{No put needed on IWN}: if you're bullish BTC and just want
        diversification (not hedging), hold IWN equity without the put
        overlay, reducing quarterly cost to zero for that leg.
  \item \textbf{SPY puts become optional}: in a strong bull BTC regime, you
        can scale SPY puts down to 5\% OTM (\$0.71/share, 67\% cheaper)
        as a tail-only insurance rather than ATM protection.
\end{itemize}
