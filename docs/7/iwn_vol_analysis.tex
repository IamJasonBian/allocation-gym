\section{Executive Summary}
% ============================================================

A review of the runtime service dashboard revealed a \textbf{critical
instrument mismatch}: the portfolio holds \textbf{Grayscale BTC Mini Trust}
(ticker BTC, \$32.35/share), not spot Bitcoin (\$72,848). This trust has
fundamentally different correlation properties than spot BTC:

\begin{itemize}[nosep]
  \item \textcolor{warn}{\textbf{Grayscale BTC is +0.76 correlated with SPY}}

        SPY puts do \textit{not} hedge this position.
  \item \textcolor{warn}{\textbf{Grayscale BTC is only +0.10 correlated with
        spot BTC/USD}} at 20d---the trust barely tracks Bitcoin itself.
  \item \textbf{Dashboard correlation method is correct}: the runtime service
        computes Pearson correlation on \textit{log returns} (not price levels).
        The high equity correlations are real, not a methodological error.
  \item \textbf{Revised hedge}: since Grayscale BTC moves with equities,
        the hedge should target equity drawdowns, not BTC-specific risk. GLD
        is the only negatively correlated asset at short windows.
\end{itemize}

\subsection{Correlation Divergence: Grayscale vs.\ Spot}

The dashboard (runtime service) computes rolling return correlations across
7 windows (5--35 trading days). The method is:
\[
  \rho_{N}(t) = \frac{\sum_{i=1}^{N}(r^{A}_i - \bar{r}^{A})(r^{B}_i - \bar{r}^{B})}
  {\sqrt{\sum(r^{A}_i - \bar{r}^{A})^2}\;\sqrt{\sum(r^{B}_i - \bar{r}^{B})^2}}
  \quad\text{where } r_i = \ln\!\left(\frac{P_i}{P_{i-1}}\right)
\]

This is the \textbf{correct} Pearson correlation on log returns.

\subsection{Implication for Hedging}

Our prior analysis (Section~\ref{sec:spot}) used \textit{spot BTC/USD}
correlations, which showed SPY as negatively correlated ($-0.112$ at 63d).
That analysis is correct for \textbf{spot Bitcoin}---but \textbf{wrong for
our actual portfolio holding}. Since Grayscale BTC moves with equities:

\begin{itemize}[nosep]
  \item \textbf{SPY puts do not hedge Grayscale BTC}---they move in the same
        direction. When SPY drops, Grayscale BTC drops too ($\rho = +0.76$).
  \item \textbf{IWN equity does not diversify Grayscale BTC}---IWM (the
        closest proxy) is $+0.77$ correlated with the trust.
  \item \textbf{GLD is the only potential hedge}: negative to low correlation
        with both Grayscale BTC and equities.
\end{itemize}
